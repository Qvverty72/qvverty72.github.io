Tech-o-computers

Tech-O-Computers is a web software project based on the Django Framework, the principal features that are used in this project are, a connection to a SQLite database, a login and register system, product catalog, shopping cart, and a full functional payment system connected to the Transbank API. Also it features a administator panel to manage the products and users in the page. This project was also developed by a team of two students in the span of three months.
